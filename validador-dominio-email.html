Validador de Domínio e MX

Para não técnicos: A ferramenta verifica se o domínio da empresa está configurado existe e está corretamente para receber e-mails de circularização. Não verificamos caixas postais individuais, apenas se o domínio tem servidores de e-mail ativos.

Processo de Validação

Como funciona esta validação?

Em linguagem simples: Verificamos se o domínio da empresa (ex: "empresa.com.br") tem servidores de e-mail contratados e configurados. É como verificar se um prédio comercial tem caixas de correio instaladas antes de enviar uma carta.

O que NÃO verificamos: Não checamos se um e-mail específico (ex: "fulano@empresa.com.br") existe ou se está ativo. Essa verificação seria como tentar abrir o cadeado de uma caixa de correio específica.

Tecnologia utilizada: Usamos a Google Public DNS API para consultar registros MX (Mail Exchange) de forma segura e confiável.
